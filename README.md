# Favorite Search App 📱

Android application built using Material Design 3.

## 📖 Description

A simple single-Activity Android app that lets users search for their favorite items.  
All data is static and hardcoded – no database, backend or API involved.  
Designed following Google's latest Material Design 3 guidelines for a clean and modern look.

## ✨ Features

- Single Activity architecture
- Modern Material Design 3 UI components (TextInputLayout, Cards, etc.)
- Scrollable screen with smooth layout
- Real-time search preview using static sample data

## 🛠 Technologies Used

- Kotlin
- XML Layouts
- Material Components (Material 3)

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/SandraKolevska/FavoriteSearchApp.git

2. Open the project in Android Studio.
3. Let Gradle sync automatically.
4. Run the app on an emulator (API 33 recommended) or a physical device.
