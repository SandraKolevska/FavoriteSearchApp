Favorite Search App 📱
Android application built using Material Design 3.

📖 Description
A simple single-Activity Android app built as part of Android coursework.

The project started as a UI-only application for Homework 1 and was later extended in Homework 2 with a working Macedonian–English dictionary that reads data from a text file.

The application follows Google's Material Design 3 guidelines for a clean and modern interface.

✨ Features
Single Activity architecture
Modern Material Design 3 UI components (TextInputLayout, MaterialButton, MaterialCardView)
Favorite Twitter Searches UI
Tagged search buttons
Dictionary search functionality (English ↔ Macedonian)
Reads dictionary data from a text file
Case-insensitive search

🛠 Technologies Used
Kotlin
XML Layouts
Material Components (Material 3)
Android Studio

🚀 How to Run
Clone the repository:

git clone https://github.com/SandraKolevska/FavoriteSearchApp.git

Open the project in Android Studio.
Let Gradle sync automatically.
Run the app on an emulator (API 33 recommended) or a physical device.

📚 Homework 2 Additions
Added a Macedonian–English dictionary feature.
The app reads words from a text file stored in the assets folder.
Users can search for a word in English or Macedonian and see the translation.

Example:

father → татко  
татко → father
